🎛️ Kontrol ON/OFF cron (71, 72, 80 & Scalev)

Halaman ini statis (GitHub Pages) jadi tombol di sini nggak boleh pegang token — kalau dipaksa, token Meta bakal bocor ke browser. Jadi kontrol cron dijalanin lewat MASBOI di grup Telegram (paling aman, nggak nyimpen kredensial di mana-mana):

Matiin cronbilang ke MASBOI: off cron a71 / off cron a72
Nyalain cronon cron a71 / on cron a72
EksekusiMASBOI komen/uncomment baris crontab di VPS, lalu kasih konfirmasi readback

Opsi tombol di dashboard bisa dibikin nanti via GitHub Actions, tapi butuh simpan SSH key sebagai GitHub Secret — nunggu keputusan Roni dulu.

📡 Status live datang dari mana?

Script sync_status.py jalan di VPS (yang pegang token), baca status campaign 1× batch per akun, lalu nulis data/a71.json & data/a72.json ke repo — tanpa token. Browser cuma baca JSON itu, nggak pernah manggil Meta langsung, jadi nggak ada spam API.

Zona waktu
